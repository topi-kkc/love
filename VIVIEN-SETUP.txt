📱 ENGLISH INSTALLATION
════════════════════════════════════════════════════════════════

1. DOWNLOAD APK
   → Download VIVIEN.apk from GitHub Releases
   → File size: ~15 MB
   → Version: 1.0.0

2. ENABLE UNKNOWN SOURCES
   → Go to Settings
   → Security & Privacy
   → Enable "Unknown Sources" or "Install Unknown Apps"

3. INSTALL APK
   → Open File Manager
   → Find VIVIEN.apk
   → Tap the file
   → Click "Install"
   → Wait for completion

4. LAUNCH APP
   → Find VIVIEN in your apps
   → Tap to open
   → Enjoy the 3D cube animation!

🆘 TROUBLESHOOTING / حل المشاكل
════════════════════════════════════════════════════════════════

PROBLEM: "Cannot install unknown apps"
SOLUTION: Enable it in Settings > Security > Unknown Sources

PROBLEM: "Installation failed"
SOLUTION: Download file again, check storage space

PROBLEM: "App crashes on startup"
SOLUTION: Restart phone, uninstall and reinstall

PROBLEM: "File not found"
SOLUTION: Download from official GitHub repository

PROBLEM: "Permission denied"
SOLUTION: Check file permissions, download from trusted source

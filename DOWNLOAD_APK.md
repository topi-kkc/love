## تحميل ملف APK

**اسم الملف**: `VIVIEN.apk`  
**الحجم**: ~15 MB  
**الصيغة**: ZIP يحتوي على APK + تعليمات التثبيت

---

## طرق التحميل

### 1️⃣ من GitHub مباشرة
1. اذهب إلى: https://github.com/topi-kkc/love
2. اضغط على "Releases"
3. حمل `VIVIEN-APK-Package.zip`
4. استخرج الملف
5. انقل `VIVIEN.apk` إلى هاتفك

### 2️⃣ من متصفح الويب
1. افتح المتصفح على هاتفك
2. اذهب إلى: https://github.com/topi-kkc/love/releases
3. ابحث عن أحدث إصدار
4. حمل `VIVIEN.apk`
5. افتح الملف للتثبيت

### 3️⃣ من خلال ADB (للمطورين)
```bash
adb install -r VIVIEN.apk
```

## خطوات التثبيت

### الخطوة 1: تفعيل المصادر المجهولة
```
الإعدادات
  ↓
الأمان والخصوصية
  ↓
التطبيقات المجهولة
  ↓
تفعيل الخيار ✓
```

### الخطوة 2: تثبيت APK
```
1. افتح مدير الملفات
2. ابحث عن VIVIEN.apk
3. اضغط على الملف
4. اضغط "تثبيت"
5. انتظر اكتمال التثبيت
```

### الخطوة 3: تشغيل التطبيق
```
1. اضغط "فتح" أو
2. اذهب إلى التطبيقات
3. ابحث عن "VIVIEN"
4. اضغط لتشغيل
```

## حل المشاكل

### ❌ "لا يمكن تثبيت تطبيقات من مصادر غير معروفة"
**الحل**: تفعيل الخيار من الإعدادات (انظر الخطوة 1)

### ❌ "الملف تالف"
**الحل**: إعادة تحميل الملف من مصدر موثوق

### ❌ "التطبيق لا يعمل"
**الحل**: إعادة تشغيل الهاتف وإعادة التثبيت

### ❌ "مساحة التخزين غير كافية"
**الحل**: حذف بعض الملفات أو التطبيقات

# ⚡ تثبيت سريع جداً - Quick Install

## العربية

### في 5 خطوات فقط:

1. **حمل الملف**
   ```
   VIVIEN.apk من GitHub Releases
   ```

2. **فعّل الخيار**
   ```
   الإعدادات → الأمان → المصادر المجهولة ✓
   ```

3. **افتح الملف**
   ```
   مدير الملفات → VIVIEN.apk
   ```

4. **اضغط تثبيت**
   ```
   تثبيت → انتظر ✓
   ```

5. **شغّل التطبيق**
   ```
   التطبيقات → VIVIEN → فتح 🎮
   ```

---

## English

### In Just 5 Steps:

1. **Download File**
   ```
   VIVIEN.apk from GitHub Releases
   ```

2. **Enable Option**
   ```
   Settings → Security → Unknown Sources ✓
   ```

3. **Open File**
   ```
   File Manager → VIVIEN.apk
   ```

4. **Tap Install**
   ```
   Install → Wait ✓
   ```

5. **Launch App**
   ```
   Apps → VIVIEN → Open 🎮
   ```

---

## مشكلة؟ Problem?

**ماذا تفعل إذا فشل التثبيت؟**
```
✓ أعد تشغيل الهاتف
✓ حمّل الملف مرة أخرى
✓ تأكد من تفعيل المصادر المجهولة
```

**What if installation fails?**
```
✓ Restart your phone
✓ Download the file again
✓ Make sure Unknown Sources is enabled
```
